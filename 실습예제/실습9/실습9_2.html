UI 디자인

UI 디자인이란 UI 아키텍처 설계 후 시각적인 요소를 디자인하기 위해 비주얼 디자인 콘셉트를 도출하며 이 콘셉트에 근거하여 GUI를 상세 디자 인하는 능력이다.

UI/UX 가이드 제작

UI/UX 가이드 제작이란 UI/UX 콘셉트, 콘텐츠, 정보 구조, 워크플로우, 서비스 정책, 디자인 산출 물을 파악하여 SW 개발을 위한 UI/UX 가이드를 제작하고 공유하는 능력이다.

UI 구현

UI 구현이란 UI 설계 산출물과 GUI 디자인 가 이드를 바탕으로 UI 구현 표준을 수립하고 UI를 제작하는 능력이다.

UI 테스트

UI 테스트란 구현된 UI를 검증하기 위하여 사 용성 테스트 계획, 수행, 분석, 결과 보고를 수행 하는 능력이다.
